# secrecy

## web
Standalone js web client.

**Settings:**
Can be found in *\web\settings.js*
(e.g. Websocket Host)

## node.js
Install node and npm.

Create a directory for your node and copy the .js files there.
In that directory, run the following commands:

### npm
```bash
    npm init --y

    npm install ws
    npm install express
    npm install uuid
```
